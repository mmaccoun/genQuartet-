This directory contains scripts used to simulate a single child's genome. The genome is simulated 
by selecting crossover locations on the maternal and paternal genome, which together compose the child's genome.
Two children must be simulated for a quartet. 

* - * File Names/Subdirectories & Descriptions * - *

simChild.py : script with the module that simulates a child's genome 

modules/ : supporting scripts with modules used in simChild.py
